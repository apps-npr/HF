🌟 เป้าหมายการรักษา HFrEF: The Quadruple Therapy

การให้ยาหลักครบทั้ง 4 กลุ่ม (ARNi/ACEi/ARB, Beta Blockers, MRAs, SGLT2i) ร่วมกันตั้งแต่แรก ลดอัตราการเสียชีวิตจากทุกสาเหตุได้ถึง 73% (เมื่อเทียบกับกลุ่มที่ไม่ได้รับการรักษา) แนะนำให้เริ่มและปรับยา (Titration) ทุก 1-2 สัปดาห์จนถึง Target dose หรือขนาดสูงสุดที่ทนได้

กลุ่มยา ประสิทธิภาพ / ระดับคำแนะนำ ขนาดยา (Starting -> Target Dose)
ARNi
(Sacubitril/Valsartan)		 Class 1, Level A: ลด Mortality และ Hospitalization 20% (เหนือกว่า ACEI)
*หากเปลี่ยนจาก ACEI ต้องหยุดยา 36 ชม. ป้องกัน Angioedema		 - ทั่วไป: 49/51 mg วันละ 2 ครั้ง
- ขนาดต่ำ (24/26 mg วันละ 2 ครั้ง): สำหรับผู้ที่ไม่เคยได้ ACEI/ARB มาก่อน, เคยได้ ACEI ขนาดต่ำ, eGFR < 30, หรือเสี่ยง BP ต่ำ
- เป้าหมาย: 97/103 mg วันละ 2 ครั้ง
ACEi / ARB Class 1, Level A: ทางเลือกเมื่อใช้ ARNi ไม่ได้ - Enalapril: 2.5 mg BID -> 10-20 mg BID
- Losartan: 25-50 mg QD -> 150 mg QD
- Valsartan: 20-40 mg BID -> 160 mg BID
Beta-blocker Class 1, Level A: ลด Mortality ชัดเจน บรรเทาอาการ - Bisoprolol: 1.25 mg QD -> 10 mg QD
- Carvedilol: 3.125 mg BID -> 25 mg BID (50mg ถ้า นน.>85kg)
- Metoprolol suc: 12.5-25 mg QD -> 200 mg QD
MRA Class 1, Level A: ลด Mortality และ Sudden Cardiac Death - Spironolactone: 12.5-25 mg QD -> 25-50 mg QD
- Eplerenone: 25 mg QD -> 50 mg QD
*ห้ามเริ่มถ้า eGFR < 30 หรือ K > 5.0
SGLT2i Class 1, Level A: ลด Hospitalization ได้ถึง 30% (ไม่ต้องเป็นเบาหวานก็ใช้ได้) - Dapagliflozin / Empagliflozin: 10 mg วันละ 1 ครั้ง
*ไม่มีการ Titration (เริ่มและคงที่ได้เลย)
Ivabradine Class 2a: ลด HF Hospitalization - เริ่ม: 5 mg วันละ 2 ครั้ง -> เป้าหมาย: 7.5 mg วันละ 2 ครั้ง
*ใช้เมื่อได้ BB เต็มที่แล้ว แต่ HR ยัง >= 70 (Sinus)
